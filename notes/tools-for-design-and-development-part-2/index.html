Tools for Design & Development Part 2

The following is a list of hardware and software I use to do my work. Tools for Design & Development (Part 1) was published on February 4, 2016.

15" Macbook Pro (2013)

  • Peripherals: 27" Apple Thunderbolt Display, Apple Magic Keyboard, and an Apple Magic Trackpad 2.
  • Desk: Team 7 Atelier Desk.
  • Chair: Eames Soft Pad Management Chair with no arms.
  • iA Writer for writing.
  • Adobe Experience Design (XD) for web design.
  • Adobe Photoshop for image editing.
  • Adobe Illustrator for vector work.
  • Terminal for command line UNIX.
  • Github Atom for writing code.
  • Panic Transmit for sFTP'ing and S3 deployments.
  • Git & Github for source control and storing repositories.
  • Chrome DevTools for debugging.
  • Google Chrome for browsing, and testing websites.
  • Firefox, and Internet Explorer for testing websites.

12.9" iPad Pro (2015)

  • Peripherals: Smart Keyboard, Apple Pencil.
  • Comic Zeal for reading comics.
  • iBooks for reading books.
  • Linea Sketch for drawing.
  • iA Writer for writing.
  • Google apps for email, documents, and maps.

iPhone Xs Max

  • Google apps for email, calendar, documents, and maps.
  • Instapaper for reading saved articles.

Apple Watch Series 4

  • Audible for audiobooks.
  • iTunes for music.
